# 🚆 Train Delay Prediction using Machine Learning

## 📌 Project Overview

Train delays are a common problem in railway transportation systems. This project focuses on predicting train delays using Machine Learning techniques. By analyzing historical train data, the model can estimate whether a train will arrive on time or be delayed.

The aim of this project is to improve travel planning for passengers and help railway systems manage schedules more efficiently.

---

## 🎯 Objectives

* Analyze historical train data.
* Identify patterns that lead to train delays.
* Build a machine learning model to predict delays.
* Evaluate model performance and accuracy.

---

## 🛠️ Technologies Used

* Pandas

* Matplotlib

---

## 📊 Dataset

The dataset used in this project contains historical train information such as:

* Train Number
* Station Name
* Scheduled Arrival Time
* Actual Arrival Time
* Delay Duration
* Distance Travelled
* Date and Time

This data is used to train the machine learning model to recognize patterns related to delays.

---

## ⚙️ Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Delay Prediction

---

## 🤖 Machine Learning Models Used

The following algorithms were implemented and tested:

* Linear Regression
* Decision Tree
* Random Forest

Among these models, the best-performing model was selected based on evaluation metrics.

---

## 📈 Results

The trained model was able to predict train delays based on historical data patterns. The model performance was evaluated using standard machine learning metrics such as accuracy and error rates.
